# FrontTrackingMethods.jl

FrontTrackingMethods.jl provides Lagrangian front-tracking for closed
2-D polygonal curves and 3-D triangulated surfaces. It builds on
[FrontIntrinsicOps.jl] for geometry, normals, curvature, and DEC operators.

This documentation provides a quick overview, installation notes, usage
examples, and focused pages on benchmarks, diagnostics, and remeshing.

- [Installation](installation.md)

- [Examples](examples.md)
- [API summary](api.md)
- [Running tests](testing.md)
- [Plotting and animations](plotting.md)
- [Benchmarks](benchmarks.md)
- [Diagnostics](diagnostics.md)
- [Redistribution & Remeshing](remeshing.md)

For more details, see the repository README and the inline docstrings in
`src/`.
